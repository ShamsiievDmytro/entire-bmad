### 2. Duplicate Unix-second timestamps crash all lightweight-charts instances

### 3. Unstable sort causes cadenceLabels/cadenceValues index mismatch
**Severity: High**

### 9. `classifyFileLayer` misclassifies paths with layer-name prefix in directory segment
**Severity: Medium**
**Location:** `src/lib/utils/ai-metrics-utils.ts:49-52`
`filePath.includes('components/')` matches any occurrence of the substring `components/` anywhere in the path, including nested or prefixed directory names like `src/ui-components/Foo.ts` (contains `components/`? No — but `src/extra-components/Foo.ts` does). More concretely, a path like `src/components-old/Foo.ts` does NOT match (no trailing slash), but `src/extra/components/Foo.ts` correctly matches.
